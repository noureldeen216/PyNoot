# PyNoot - بايثون بالعربي 🇪🇬🐍

مكتبة Python بالعربي لتعليم البرمجة بسهولة وبلغة مألوفة للمبتدئين العرب.

## المميزات:
- `اطبع()` بدل `print()`
- `ادخل()` بدل `input()`
- `لو()` بدل `if`
- `امسح()` لمسح الشاشة
- والمزيد قريبًا!
