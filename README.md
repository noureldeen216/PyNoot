# 🐍 باي نطق (PyNoot)
### "البرمجة مش لغة صعبة.. البرمجة منطق بنعبر عنه بلغتنا"

---

## ✨ إيه هو مشروع "باي نطق"؟
**باي نطق** مش مجرد أداة بتخليك تكتب كلمات بالعربي وخلاص، هي "كوبري" حقيقي بيدخلك عالم البرمجة من أوسع أبوابه. 

الهدف الأساسي من **باي نطق** هو إننا نشيل عائق اللغة في البداية، ونخليك تركز في الحاجة الأهم وهي: **فهم المنطق البرمجي (Logic)**. إحنا بنعلمك إزاي تفكر كمبرمج، إزاي ترتب خطواتك، وإزاي تتعامل مع البيانات بأسلوب بسيط وقريب من دماغك وبالمصري.

---

## 🚀 ليه "باي نطق" مختلف؟
* **التركيز على المنطق (Logic First):** مش بنعلمك تحفظ أوامر، بنعلمك "ليه" بنكتب الكود ده وإزاي بيشتغل كمنطق هندسي سليم.
* **أسلوب صحاب (Friendly):** المكتبة دي معمولة بروح ودودة جداً، الأوامر والرسائل فيها واضحة ومباشرة كأن حد قاعد بيشرحلك بعيداً عن كلكعة المصطلحات.
* **كل الأساسيات في موديول واحد:** ميزة **باي نطق** إنها بتجمعلك أهم أساسيات بايثون (Python Basics) اللي محتاجها عشان تبدأ حياتك البرمجية جوه **Module** واحد منظم.
* **دليل تعليمي شامل (PDF Documentation):** المشروع معاه ملف PDF احترافي فيه شرح لكل كبيرة وصغيرة بالصور وبالعربي، عشان يفضل معاك مرجع في كل خطوة.

---

## 🛠️ الأوامر ومقابلها في بايثون (Dictionary)
عشان نسهل عليك النقل للغة بايثون الأصلية بعد كدة، دي الأوامر اللي جوه **باي نطق** وايه اللي بيقابلها في بايثون:

| الأمر في "باي نطق" | الأمر الأصلي في Python | وظيفة الأمر |
| :--- | :--- | :--- |
| `اطبع()` | `print()` | عرض النصوص أو النتائج على الشاشة |
| `نطق()` | `input()` | استقبال بيانات من المستخدم |
| `نوع()` | `type()` | معرفة نوع البيانات (نص، رقم، إلخ) |
| `طول()` | `len()` | معرفة عدد العناصر أو الحروف |
| `رتب()` | `sort()` | ترتيب العناصر في القائمة |
| `ضيف_عنصر()` | `append()` | إضافة عنصر جديد لآخر القائمة |
| `احذف_عنصر()` | `pop()` | حذف عنصر معين من القائمة |
| `امسح_الكل()` | `clear()` | مسح كل محتويات القائمة |
| `شقلب()` | `reverse()` | عكس ترتيب عناصر القائمة |
